# InveShop
## Inveon FullStack Bootcamp - Final Project
InveShop is an e-commerce project.

### Tech Stack
- .NET 7
- React
- Ocelot
- Microsoft SQL Server
- RabbitMQ
- Entity Framework Core
- AutoMapper
- Iyzipay
- OpenID
- Razor Templating Core
- Imagekit
- Jwt Bearer
- Duende Identity Server
- Redux
- Axios
- Google Maps API
